Online Quiz Application

A complete walkthrough of how this system works

Overview

This application allows users to take quizzes online. Here's the process in simple terms:

  1. Admin adds questions to the question bank.
  2. Users take the quiz by answering questions from the bank.
  3. The system calculates their score automatically.
  4. The score and results are stored for future reference.

Two main data files power this app: questions.json and results.json.

Detailed Specification

Objective

Conduct multiple-choice quizzes and calculate results.

Core Classes

  • Question – question text, options (list), correct answer.
  • Quiz – manages list of questions and scoring system.

Core Features

  1. Add questions to the quiz bank.
  2. Conduct quiz (show questions, take answers).
  3. Calculate and display final score.

Example Flow

  1. Add Q: “Which is a Python data type? (A: list, B: table, C: chart)” → Correct: A.
  2. User attempts quiz → answers stored.
  3. Score shown: “You got 4/5 correct.”
  4. Result saved in results.json.

1. Question Bank Management

This module stores all quiz questions and allows adding, editing, or deleting them.

  • Question Text – e.g., What is the capital of France?
  • Options – Multiple choice answers.
  • Correct Answer – Used for score calculation.
  • Category – e.g., General Knowledge, Science, etc.

All questions are stored in questions.json.

2. Quiz Taking

When a user takes a quiz:

  1. The system loads questions from questions.json.
  2. Questions are displayed one by one or all at once.
  3. User selects their answers.
  4. Navigation controls like Next and Previous may be provided.
  5. A timer can be used for time-bound quizzes.

3. Score Calculation

When the quiz is submitted:

  1. The system compares the user's answers with correct answers from questions.json.
  2. Each correct answer increases the score.
  3. The final score is displayed instantly.

4. Result Storage

After score calculation:

  1. User's name, score, and quiz date are recorded.
  2. This data is saved in results.json.
  3. Stored results can be used to track progress over time.

JSON File Structures

questions.json

[ { "question": "What is the capital of France?", "options": ["Paris", "London", "Berlin", "Rome"], "answer": "Paris", "category": "General Knowledge" } ]

results.json

[ { "username": "Shivam Bansal", "score": 8, "date": "2025-08-13" }, { "username": "Mohini Bansal", "score": 6, "date": "2025-08-13" } ]
