Library Membership & Book Issue System

Manage members, books, borrowing, and returns

Overview

This system helps manage library books, track memberships, handle borrowing & returning, and generate reports.

  1. Register new members with personal details.
  2. Add and manage book inventory.
  3. Issue books to members with due dates.
  4. Record returns and update availability.
  5. Generate reports of borrowed and returned books.

Data is stored in books.json, members.json, and borrowHistory.json.

Detailed Specification

Objective

Manage library members, books, and borrowing with due date tracking.

Core Classes

  • Book – title, author, ISBN, availability.
  • Member – member ID, name, contact info, borrowed books list.
  • Library – manages collections of books and members.

Core Features

  1. Add/remove books from the library.
  2. Register members.
  3. Borrow/return books (update availability).

Example Flow

  1. Admin adds “Data Structures in Python” to library.
  2. Student A registers as a member.
  3. Student A borrows the book → status updated to “unavailable”.
  4. After 10 days (due was 7 days), fine is calculated.

1. Book Management

When adding new books, record the following:

  • Book Title – Name of the book.
  • Author – Author's name.
  • Availability – Available or borrowed.

Saved in books.json for tracking inventory.

2. Member Registration

When a new member joins, record the following:

  • Member ID – Unique identifier (e.g., LIB101).
  • Name – Full name of the member.
  • Contact Info – Phone and/or email.

Stored in members.json for library records.

3. Borrow & Return

Borrow process:

  1. Select available book and member.
  2. Set due date.
  3. Mark as borrowed in borrowHistory.json.

Return process:

  1. Find matching borrow record.
  2. Mark as returned and update availability.

4. Reports

Generate reports for:

  • All currently borrowed books.
  • Overdue books.
  • Borrow history per member.

JSON File Structures

books.json

[ { "title": "The Great Gatsby", "author": "F. Scott Fitzgerald", "available": true } ]

members.json

[ { "member_id": "LIB101", "name": "Anita Verma", "contact": "anita@example.com" } ]

borrowHistory.json

[ { "memberName": "Anita Verma", "bookTitle": "The Great Gatsby", "dueDate": "2025-08-20", "returned": false } ]
