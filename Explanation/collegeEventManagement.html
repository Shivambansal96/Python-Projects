College Event Registration System

Manage events, participants, and reports for a college fest

Overview

This system is designed to handle multiple events in a college fest, from managing event details to registering participants and generating reports.

  1. Create and manage events with details such as date and venue.
  2. Register participants for one or more events.
  3. Store data securely in JSON or CSV files.
  4. Search participants by event name.
  5. Generate summary reports for each event.

Detailed Specification

Objective

A system to manage college fest events, register participants, and track registrations.

Core Classes

  • Event – event name, description, date, capacity, registered participants list.
  • Participant – name, roll number, department, registered events list.
  • EventManager – manages event creation, deletion, participant registration.

Core Features

  • Add, update, delete events.
  • Register participants for events.
  • Prevent registration if the event is full.
  • Search for participants by event name.

Example Flow

  1. Admin adds “Coding Contest”, capacity 30.
  2. Student registers for “Coding Contest”.
  3. System checks availability → confirms registration.
  4. Admin searches “Coding Contest” → sees participant list.

1. Event Management

Using the Event class, you can:

  • Add new events with an Event ID, name, date, and venue.
  • Update existing event details.
  • Delete events if needed.

All events are stored in events.json or events.csv.

2. Participant Registration

Using the Participant class, you can:

  • Register students with name, department, and event participation.
  • Assign a unique Participant ID.
  • Allow a participant to register for multiple events.

Participant details are stored in participants.json or participants.csv.

3. File Storage

Two storage formats can be used:

  • JSON – Easy for storing and retrieving structured data.
  • CSV – Easy for spreadsheet viewing and sharing.

4. Search & Reporting

Features include:

  • Search participants by event name or ID.
  • Generate reports showing all participants for an event.
  • Count the number of participants per event.

Sample File Structures

events.json

[ { "event_id": "EV001", "name": "Tech Quiz", "date": "2025-09-15", "venue": "Auditorium" } ]
